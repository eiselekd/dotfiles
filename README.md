dotfiles
========

Emacs config.

Keybindings:

 * ESC g : magit
 * ESC c : flymake
 * F5/F6 : next/prev-error
 * F10   : compile
   * compile command from "compile.txt"
 * F11   : gdb
   * gdb init script from  "gdb.txt"
   * F1 : restore layout
   * F2 : step into
   * F3 : step over
   * F4 : continue
   * F12: set breakpoint
 * ALT-Shift up/down/left/right : resize window
 * Shift up/down/left/right : jump window
